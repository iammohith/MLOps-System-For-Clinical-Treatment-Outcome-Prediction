# Data Directory

This directory contains the raw and processed data for the Clinical Treatment Outcome Prediction system.

## Structure

```
data/
├── raw/
│   └── real_drug_dataset.csv    # Original dataset (DVC tracked)
└── processed/                    # Generated by preprocessing pipeline
    ├── ingested.csv
    ├── validated.csv
    ├── X_train.csv
    ├── X_test.csv
    ├── y_train.csv
    ├── y_test.csv
    └── preprocessor.joblib
```

## Dataset

- **Source**: [Kaggle — 1000 Drugs and Side Effects](https://www.kaggle.com/datasets/palakjain9/1000-drugs-and-side-effects)
- **Rows**: 1,000 patients
- **Target**: `Improvement_Score` (0–10 scale)

## Schema

| Column | Type | Description |
|--------|------|-------------|
| Patient_ID | string | Pseudonymized identifier |
| Age | int | 18–79 |
| Gender | string | Female, Male |
| Condition | string | 5 conditions |
| Drug_Name | string | 15 drugs |
| Dosage_mg | float | 50, 100, 250, 500, 850 |
| Treatment_Duration_days | int | 5–59 |
| Side_Effects | string | 30 side effects |
| Improvement_Score | float | Target (2.5–10.0) |
